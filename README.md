## 📋 О проекте

Серверная часть информационной системы для автоматизации управления складскими операциями торгового предприятия. Система предоставляет RESTful API для ведения учета товаров, контроля остатков, управления поставками и отгрузками.

## ✨ Функциональные возможности

### 📦 Управление товарами
- Каталог товаров с полной информацией
- Категоризация и поиск товаров
- Учет остатков в реальном времени

### 📊 Складской учет
- Прием и отгрузка товаров
- Инвентаризация запасов
- Отслеживание срока годности
- Управление ячейками хранения

### 🔄 Бизнес-процессы
- Обработка заказов поставщиков
- Управление заказами клиентов
- Формирование отчетов
- Аналитика продаж

### 👥 Пользователи и права
- Многоуровневая система ролей (администратор, кладовщик, менеджер)
- Разграничение прав доступа
- История действий пользователей

## 🛠 Технологический стек

| Компонент | Технология |
|-----------|------------|
| **Backend Framework** | Laravel 10+ |
| **Database** | MySQL 8.0+ |
| **API** | RESTful API |
| **Authentication** | Laravel Sanctum |
| **Validation** | Laravel Validator |
| **Caching** | Redis |
| **Queue** | Laravel Queues |
| **Testing** | PHPUnit |
